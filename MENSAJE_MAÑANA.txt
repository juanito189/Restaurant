¡Hola equipo! 

Aquí está el repositorio con todo listo para empezar. Sigan esto paso a paso:

## 🚀 PASO 1: Clonar el Repo (5 min)

1. Abren una terminal (PowerShell/CMD en Windows, Terminal en Mac/Linux)
2. Copian y pegan esto TODO DE UNA VEZ:

### Si eres EQUIPO SENA (Frontend):
```
git clone https://github.com/juanito189/Restaurant.git
cd Restaurant
python -m venv entorno_restaurante
entorno_restaurante\Scripts\Activate.ps1
pip install -r requirements.txt
git checkout feature/sena
```

### Si eres EQUIPO CESDE (Backend):
```
git clone https://github.com/juanito189/Restaurant.git
cd Restaurant
python -m venv entorno_restaurante
entorno_restaurante\Scripts\Activate.ps1
pip install -r requirements.txt
git checkout feature/cesde
```

## 🗄️ PASO 2: Configurar Base de Datos (Solo CESDE) (10 min)

*Si eres SENA, brinca este paso*

1. Abren OTRA terminal
2. Escriben: `mysql -u root -p`
3. Ingresan su contraseña de MySQL
4. Pegan esto:
```
CREATE DATABASE restaurante;
USE restaurante;
source backup_datos.sql
```

## ✅ PASO 3: Verificar que Funciona (5 min)

En la terminal del proyecto (la primera):
```
python manage.py migrate
python manage.py runserver
```

Abren el navegador a: http://localhost:8000

Si ven la página del restaurante, ¡LISTO! ✅

## 🧠 PASO 4: Abrir en Visual Studio Code

1. Abren VS Code
2. File → Open Folder → Eligen la carpeta Restaurant
3. Esperan a que cargue

## 🤖 PASO 5: Usar el Agente (LO IMPORTANTE)

1. Presionan: **Ctrl + Shift + I**
2. Se abre un chat a la derecha
3. Escriben: "Hola, soy del Equipo [SENA/CESDE], ¿qué hago?"
4. El agente Lee este README y TE GUÍA PASO A PASO

El agente entiende el proyecto, sabe tu rol, y te ayuda a resolver problemas.
